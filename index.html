Diseño de Sitios Web - OVI

GIT/GITHUB, HTML5 y CSS3

Código del Curso:

301122

Programa:

Ingeniería de Sistemas

Breve descripción de la temática general:

Introducción, profundización e información complementaria sobre el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, HTML5 y CSS3 para creación de contenidos en formato web (Paginas y Sitios Web)

Objetivo general:

Implementar el control básico de versiones GIT y GITHUB en una pequeña pagina web utilizando los conceptos básicos de HTML5 y CSS.
  • Objetivo específico 1: Comprender qué es un sistema control de versión y cuáles son las ventajas de utilizarlo en el día a día del diseño web.

  • Objetivo específico 2: Implementar los conceptos básicos de HTML5 como código de desarrollo web, comprendiendo las etiquetas como la estructura principal del contenido.

  • Objetivo específico 3:Aprender a crear hojas de estilo en CSS3 y heredarlo en las etiquetas, para estructuración visual de un cuerpo html.
